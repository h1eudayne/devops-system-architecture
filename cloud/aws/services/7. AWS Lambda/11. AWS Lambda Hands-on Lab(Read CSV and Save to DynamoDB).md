# 11. AWS Lambda Hands-on Lab(Read CSV and Save to DynamoDB) (Lab đọc CSV lưu vào DynamoDB)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**lab**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Tạo bảng employee trên DynamoDB với Partition Key id và Sort Key name.
2. Tạo Lambda function csv-to-dynamodb và cấu hình Memory 512 MB, Timeout 2 phút.
3. Cấp quyền cho Lambda bằng cách gắn các policy AmazonDynamoDBFullAccess và AmazonS3FullAccess vào IAM Role.
4. Cấu hình S3 Event Trigger kích hoạt Lambda khi tải tệp CSV lên thư mục employee_csv/ với đuôi .csv.
5. Tạo tệp CSV kiểm thử nhân viên (employee.csv) và xác nhận dữ liệu tự động lưu vào bảng DynamoDB.
