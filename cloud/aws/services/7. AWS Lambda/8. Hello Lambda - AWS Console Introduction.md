# 8. Hello Lambda - AWS Console Introduction (Làm quen với AWS Lambda Console)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**deploy**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Đăng nhập vào AWS Console và chọn tùy chọn tạo hàm Lambda mới.
2. Thiết lập cấu hình tên hàm `hello-world` và chọn runtime Python.
3. Tạo và cấu hình sự kiện kiểm thử (**Test Event**).
4. Viết mã nguồn Python trả về Hello World và thực hiện Deploy code.
5. Thực thi kiểm thử và xem chi tiết kết quả chạy thành công.
