# 10. AWS Lambda Hands-on Lab(EC2 Auto Start-Stop) (Lab bật tắt EC2 tự động)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**lab**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Tạo máy chủ ảo EC2 để lấy Instance ID thử nghiệm.
2. Tạo Lambda Function và triển khai mã nguồn Python điều khiển EC2 dựa trên tham số.
3. IAM Policy cấp quyền cho phép Lambda gọi lệnh Start/Stop Instance.
4. Lập lịch tự động thông qua EventBridge Scheduler và truyền payload JSON chứa tham số `action` và `instance_id`.
5. Kiểm tra log thực thi trên CloudWatch Logs để xác minh hoạt động.
