# 10. AWS Lambda Hands-on Lab(EC2 Auto Start-Stop) (Lab bật tắt EC2 tự động)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**lab**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Tạo IAM Policy cấp quyền điều khiển EC2 cho Lambda (Describe, Start, Stop).
2. Tạo Lambda Function chạy với Python và gán Role điều khiển EC2.
3. Viết mã nguồn Python sử dụng thư viện `boto3` để tìm kiếm các instances có Tag cụ thể (ví dụ: `Env: Dev`) và gửi lệnh Bật/Tắt.
4. Tạo cấu hình test event và chạy thử để kiểm nghiệm Bật/Tắt EC2 trực tiếp trên AWS Console.
5. Cấu hình lập lịch tự động qua Amazon EventBridge Rules để bật máy chủ vào buổi sáng (07:00) và tắt máy chủ vào buổi tối (19:00) từ Thứ 2 đến Thứ 6.
