# ECS (Elastic Container Service)

## I. Tổng quan về Amazon ECS

**Nó là gì:**
ECS là dịch vụ container orchestration của AWS, cho phép chạy Docker container mà không cần tự quản lý Kubernetes. ECS hỗ trợ hai launch type: EC2 (tự quản lý instance) và Fargate (serverless, AWS quản lý infra). ECS tích hợp chặt chẽ với ALB, IAM, CloudWatch và các dịch vụ AWS khác.

**Khi nào sử dụng:**
* Khi cần chạy container đơn giản mà không cần độ phức tạp của Kubernetes.
* Khi muốn sử dụng Fargate để chạy serverless container, không cần quản lý server.
* Khi toàn bộ hạ tầng đã nằm trên AWS và cần tích hợp native.

---

## II. Hướng dẫn cơ bản để bắt đầu

1. Tạo ECS Cluster trên AWS Console hoặc Terraform.
2. Tạo Task Definition định nghĩa container image, CPU, memory, port mapping.
3. Tạo Service để chạy và duy trì số lượng task mong muốn.
4. Gắn Application Load Balancer (ALB) để phân phối traffic.
5. Cấu hình Auto Scaling cho service theo CPU/memory utilization.
