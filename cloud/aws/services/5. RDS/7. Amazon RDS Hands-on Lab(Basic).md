# Amazon RDS Hands-on Lab (Basic)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**deploy**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Đăng nhập vào AWS Console và chọn tùy chọn cấu hình đầy đủ (Full Configuration) để tạo cơ sở dữ liệu.
2. Thiết lập cấu hình cơ sở dữ liệu với Database Engine MySQL.
3. Lựa chọn cấu hình phần cứng RDS Instance sử dụng lớp `db.t3.medium`.
4. Cấu hình bảo mật mạng nâng cao (chặn Public access) và khởi tạo DB Instance.
5. Cấu hình mở cổng (Port) kết nối tại Security Group (cổng 3306).
6. Kết nối tới RDS Instance bằng công cụ Database Client (DBeaver).
7. Thực thi mã lệnh SQL (CRUD) và Giả lập Slow Query qua script mẫu.
8. Khởi tạo thêm 1 RDS Instance đóng vai trò Read Replica và lưu ý sự khác biệt của các Endpoint.
9. Kiểm nghiệm tính chất Chỉ Đọc (Read-Only) của Read Replica bằng các lệnh Insert, Update, Delete.
10. Xóa Master Instance và kiểm nghiệm cơ chế tự động thăng cấp (Promotion/Failover) của Read Replica.
