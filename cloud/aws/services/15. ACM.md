# ACM (AWS Certificate Manager)

## I. Tổng quan về AWS Certificate Manager

**Nó là gì:**
ACM là dịch vụ quản lý TLS/SSL certificate miễn phí của AWS. ACM tự động cấp và gia hạn certificate cho các domain của bạn. Certificate từ ACM có thể gắn vào ALB, CloudFront, API Gateway mà không cần tự quản lý file cert/key.

**Khi nào sử dụng:**
* Khi cần HTTPS cho ứng dụng phía sau ALB hoặc CloudFront.
* Khi muốn tự động gia hạn certificate mà không cần thao tác thủ công.
* Khi cần wildcard certificate (ví dụ: `*.myapp.com`) cho nhiều subdomain.

---

## II. Hướng dẫn cơ bản để bắt đầu

1. Request certificate trong ACM Console hoặc bằng AWS CLI:
   ```bash
   aws acm request-certificate --domain-name myapp.com --subject-alternative-names *.myapp.com --validation-method DNS
   ```
2. Tạo DNS validation record trong Route 53 (ACM sẽ hiển thị chi tiết record cần tạo).
3. Đợi ACM xác nhận và cấp certificate (thường mất vài phút với DNS validation).
4. Gắn certificate vào ALB Listener (HTTPS port `443`) hoặc CloudFront Distribution.
5. **Lưu ý:** Đối với CloudFront, certificate bắt buộc phải được tạo ở region `us-east-1`.
