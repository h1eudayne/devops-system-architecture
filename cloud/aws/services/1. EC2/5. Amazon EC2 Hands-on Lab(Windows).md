# Amazon EC2 Hands-on Lab (Windows)

Bài học thực hành này đã được hợp nhất và chuyển sang thư mục hướng dẫn triển khai (**deploy**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Khởi tạo thực tế máy chủ ảo EC2 từ AMI Windows Server 2022 Base.
2. Lưu ý quan trọng về dung lượng ổ cứng tối thiểu của hệ điều hành Windows là 30 GiB.
3. Cấu hình luật tường lửa Security Group (mở cổng RDP 3389 chỉ từ IP cá nhân).
4. Sử dụng Private Key đã tạo từ trước để giải mã mật khẩu mặc định của tài khoản `Administrator`.
5. Sử dụng ứng dụng Remote Desktop Connection trên Windows để truy cập và quản trị giao diện đồ họa của máy chủ.
