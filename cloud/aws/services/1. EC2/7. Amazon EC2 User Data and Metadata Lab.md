# Amazon EC2 User Data và Metadata Lab

Bài học thực hành này đã được hợp nhất và chuyển sang thư mục hướng dẫn triển khai (**deploy**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Khởi tạo thực tế máy chủ ảo EC2 từ AMI Amazon Linux 2.
2. Cấu hình kịch bản tự động hóa **User Data** trong mục Advanced Details khi tạo máy chủ.
3. Sử dụng kịch bản tự động để tải và cài đặt Apache Web Server (httpd) khi khởi động.
4. Lấy Session Token bảo mật của **IMDSv2** bằng phương thức PUT.
5. Truy xuất các thông số **Private IP** (`local-ipv4`) và **Public IP** (`public-ipv4`) động từ máy chủ vật lý vật chủ để ghi đè vào mã nguồn trang chủ `index.html`.
6. Truy cập kiểm tra trang web hiển thị địa chỉ IP động thực tế.
