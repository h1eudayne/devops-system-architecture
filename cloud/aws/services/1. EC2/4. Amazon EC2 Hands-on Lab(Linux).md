# Amazon EC2 Hands-on Lab (Linux)

Bài học thực hành này đã được hợp nhất và chuyển sang thư mục hướng dẫn triển khai (**deploy**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Khởi tạo thực tế máy chủ ảo EC2 từ AMI Amazon Linux 2.
2. Tạo Key Pair bảo mật và cấu hình luật tường lửa Security Group (mở SSH và HTTP).
3. Cách khắc phục lỗi quyền truy cập tệp tin `.pem` đặc thù của hệ điều hành Windows bằng Advanced Security Properties.
4. Cài đặt phần mềm máy chủ Apache (httpd) và khởi tạo trang web tĩnh cơ bản.
5. Sao lưu dữ liệu ổ đĩa bằng Snapshot và tạo ảnh đĩa hệ thống AMI để nhân bản máy chủ.
