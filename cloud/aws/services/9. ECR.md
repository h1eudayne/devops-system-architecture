# ECR (Elastic Container Registry)

## I. Tổng quan về Amazon ECR

**Nó là gì:**
ECR là dịch vụ Docker container registry được quản lý bởi AWS. Nó cho phép lưu trữ, quản lý và deploy Docker image một cách an toàn. ECR hỗ trợ image scanning để phát hiện lỗ hổng bảo mật, lifecycle policy để tự động dọn dẹp image cũ, và tích hợp trực tiếp với EKS, ECS.

**Khi nào sử dụng:**
* Khi cần private Docker registry trên AWS để lưu trữ image cho pipeline CI/CD.
* Khi muốn quét bảo mật (vulnerability scanning) cho image trước khi deploy.
* Khi sử dụng EKS hoặc ECS và cần pull image nhanh trong cùng region.

---

## II. Hướng dẫn cơ bản để bắt đầu

1. Tạo repository:
   ```bash
   aws ecr create-repository --repository-name my-app
   ```
2. Login Docker vào ECR:
   ```bash
   aws ecr get-login-password --region ap-southeast-1 | docker login --username AWS --password-stdin <account-id>.dkr.ecr.ap-southeast-1.amazonaws.com
   ```
3. Build và tag image:
   ```bash
   docker build -t my-app .
   docker tag my-app:latest <ecr-uri>:latest
   ```
4. Push image:
   ```bash
   docker push <ecr-uri>:latest
   ```
5. Cấu hình lifecycle policy để tự động xóa image cũ (giữ lại 10 image gần nhất).
