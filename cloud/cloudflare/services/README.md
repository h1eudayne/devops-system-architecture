# Cloudflare Services

Thư mục này chứa tài liệu hướng dẫn và cấu hình cho các dịch vụ cốt lõi của Cloudflare.

## Danh sách dịch vụ

1. **[1. DNS Management](./1.%20DNS/1.%20Cloudflare%20DNS%20Management.md)**
   * Hướng dẫn cấu hình bản ghi DNS A để trỏ domain phụ về IP Public.
2. **[2. Zero Trust Tunnel](./2.%20Zero%20Trust/2.%20Cloudflare%20Zero%20Trust%20Tunnel.md)**
   * Hướng dẫn thiết lập Cloudflare Tunnel (Zero Trust) để kết nối dịch vụ nội bộ mà không cần mở port modem.
